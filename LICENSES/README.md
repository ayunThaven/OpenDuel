# OpenDuel licensing

OpenDuel uses multiple licenses depending on the type of content.

## Firmware and software

Files related to firmware, software tools and source code are licensed under the MIT License.

See the root `LICENSE` file.

Affected areas include:

- `firmware/`
- future software tools
- future dashboard code
- future scripts

## Documentation

Documentation files are licensed under the Creative Commons Attribution 4.0 International License.

Affected areas include:

- `docs/`
- documentation sections of `README.md`
- assembly guides
- development guides

## Hardware and mechanical files

Hardware, electronics and mechanical design files are licensed under the CERN Open Hardware Licence Version 2 - Permissive.

Affected areas include:

- `hardware/`
- `mechanical/`
- CAD files
- PCB files
- printable files
- electronics design files

## Licensed assets

OpenDuel must not distribute official licensed assets from third-party franchises.

This includes, but is not limited to:

- official logos;
- card artwork;
- copyrighted illustrations;
- proprietary visual assets;
- official product designs copied as-is.

OpenDuel is an independent fan-made open-source project.
